## Install

```bash
git clone https://github.com/whaleen/jekyll-tachyons
```

```bash
cd jekyll-tachyons
```

```bash
bundle
```

## Serve

```bash
bundle exec jekyll serve
```

## BrowserSync

https://www.browsersync.io
```bash
npm install -g browser-sync
```

```bash
browser-sync start --proxy "localhost:4000" --files "_site/*.*"
```
